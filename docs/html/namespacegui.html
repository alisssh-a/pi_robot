Класс визуализации игрового процесса

Внутреннее окно приложения для отображения игровой визуализации

Отрисовывает робота и цель на игровом поле, обрабатывает пользовательский ввод и обеспечивает плавную анимацию с помощью таймера перерисовки

Обеспечивает управление окном игры, интеграцию с системой локализации и корректное завершение работы визуализатора при закрытии
